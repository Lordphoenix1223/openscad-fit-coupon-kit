# Design Notes

## Philosophy

The point of a coupon is not to look nice. The point is to fail cheaply.

## Recommended Order

1. Check hardware-hole fit first.
2. Check slot widths second.
3. Check sliding clearance third.
4. Only then trust a full assembly print.

## Defaults

- M3 clearance test centers around `3.4 mm`
- rough PLA sliding clearance starts around `0.30-0.40 mm` per side
- slot tests should cover more than one width because strap material lies
