# Security Policy

This repository contains OpenSCAD source and documentation only. It should not contain secrets, provider credentials, local machine paths, or unpublished project material.

## Reporting

- Prefer GitHub private vulnerability reporting if it is enabled for this repository.
- If private reporting is not available, open a minimal issue requesting contact and keep the report high level.
- Do not post secrets, internal paths, or sensitive screenshots in issues or pull requests.

## Scope Notes

- There is no backend, auth layer, API surface, or network behavior in this repository.
- Publication safety and source hygiene matter more here than classic web-app controls.
