# Recommended Starting Values

## M3 Hardware

- tight test: `3.2 mm`
- practical clearance: `3.4 mm`
- generous first-print check: `3.6 mm`

## Sliding Fits

- optimistic: `0.25 mm` per side
- realistic service-print starting point: `0.35 mm` per side
- sloppy but safe: `0.45 mm` per side

## Strap / Band Slots

Print more than one width. The "obvious" slot width is often wrong once the real material bends, frays, or stacks.
